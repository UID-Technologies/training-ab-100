## Lab 03 - Cost benefit analysis
**cost-benefit analysis / ROI for an AI-powered business solution** as:

> “Before building an AI solution, we must prove that the business value is higher than the total cost of building, running, governing, and supporting it.”

## 1. Start with the business problem

Example:

ABC Manufacturing receives **10,000 support tickets per month**.

Current situation:

| Metric                  |       Value |
| ----------------------- | ----------: |
| Tickets/month           |      10,000 |
| Avg handling time       |  12 minutes |
| Support agent cost/hour |        ₹600 |
| Monthly manual effort   | 2,000 hours |
| Monthly support cost    |  ₹12,00,000 |

Calculation:

```text
10,000 tickets × 12 minutes = 120,000 minutes
120,000 / 60 = 2,000 hours
2,000 × ₹600 = ₹12,00,000 per month
```

---

## 2. Estimate AI improvement

Assume AI agent handles **40% of tickets automatically**.

| Metric                |       Value |
| --------------------- | ----------: |
| Automated tickets     | 4,000/month |
| Time saved per ticket |  12 minutes |
| Monthly hours saved   |   800 hours |
| Monthly labor savings |   ₹4,80,000 |

Calculation:

```text
4,000 × 12 minutes = 48,000 minutes
48,000 / 60 = 800 hours
800 × ₹600 = ₹4,80,000 monthly savings
```

---

## 3. Identify costs

AI solution cost includes more than model cost.

| Cost Type           | Example                                 |
| ------------------- | --------------------------------------- |
| Platform cost       | Copilot Studio, Azure AI Foundry        |
| Model/token cost    | GPT model usage                         |
| Licensing           | Microsoft 365 Copilot, Power Platform   |
| Development cost    | Design, build, integration              |
| Integration cost    | APIs, connectors, Dynamics, ServiceNow  |
| Data preparation    | Knowledge cleanup, grounding, vector DB |
| Security/governance | RBAC, audit, responsible AI             |
| Support cost        | Monitoring, tuning, maintenance         |

Example monthly cost:

| Cost Item                 | Monthly Cost |
| ------------------------- | -----------: |
| Copilot Studio / platform |    ₹1,00,000 |
| Azure AI model usage      |      ₹50,000 |
| Monitoring/support        |      ₹75,000 |
| Knowledge maintenance     |      ₹25,000 |
| Total monthly cost        |    ₹2,50,000 |

---

## 4. Calculate net benefit

```text
Monthly benefit = ₹4,80,000
Monthly AI cost = ₹2,50,000

Net monthly benefit = ₹4,80,000 - ₹2,50,000
Net monthly benefit = ₹2,30,000
```

---

## 5. Calculate ROI

Formula:

```text
ROI % = (Net Benefit / Total Cost) × 100
```

Example:

```text
ROI = (₹2,30,000 / ₹2,50,000) × 100
ROI = 92%
```

This means:

> For every ₹100 spent on the AI solution, the organization receives ₹92 net return in that month.

---

## 6. Calculate payback period

Assume one-time implementation cost:

```text
Implementation cost = ₹12,00,000
Net monthly benefit = ₹2,30,000
```

Formula:

```text
Payback period = Implementation cost / Monthly net benefit
```

```text
₹12,00,000 / ₹2,30,000 = 5.2 months
```

Meaning:

> The AI project recovers its implementation cost in around 5 months.

---

# Explanation

You can explain it like this:

```text
For any AI-powered solution, ROI is not calculated only from model cost.

We compare the total value generated by AI against the total cost of ownership.

Value may come from time saved, reduced manual effort, faster response, better accuracy, improved customer experience, fewer errors, and increased revenue.

Cost includes model usage, licenses, platform, integration, data preparation, governance, monitoring, and support.

If benefits are higher than costs, and payback period is acceptable, the AI solution is financially justified.
```



# Key Point

> AI ROI is not just token cost.
> AI ROI = business outcome + productivity gain + risk reduction + customer impact − total cost of ownership.
